# Exploratory Data Analysis: Soccer Shooting Statistics

**Overview**

This project explores soccer shooting data to analyze player efficiency, focusing on Goals per Shot on Target and Shot on Target Percentage (SoT%). The goal is to uncover insights into which players are most effective when their shots are on target.

**Dataset**

The dataset contains player-level shooting statistics, including:

• Shot distance

• Shots on target

• Goals scored

• Player position and team

**For detailed analysis, see the EDA1.docx file.**
